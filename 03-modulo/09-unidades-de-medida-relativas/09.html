Unidades de medida relativas (em oposição às unidades de medida absolutas).
A unidade de medida "porcentagem (%)" determina o tamanho de um elemento DE ACORDO COM O TAMANHO CONFIGURADO DO ELEMENTO PAI.
No exemplo abaixo, o elemento "container" está configurado para ter 80% do tamanho de seu elemento pai (o body - que está relacionado a toda a extensão da tela do dispositivo).
De acordo com a mesma lógica, o elemento "inside" está programado para ter 80% do tamanho de seu elemento pai (o container).
